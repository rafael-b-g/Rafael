This curiosity led me to a passion for STEM subjects. However, I'm not just a techie person; I also love design. The intersection of arts and technology is where things get interesting, and that's where I am.

Tech.

These are some of the technologies that I use and I'm familiar with:

Dart

Dart is a programming language used to create apps (with Flutter). The syntax is very similar to Javascript.

Flutter

Flutter is a framework (a set o tools) that uses Dart to create apps for many platforms with the same source code.

Javascript

Javascript (JS) is the most used programming language on the web. It is used on practically every website out there.

HTML and CSS

HTML and CSS are languages used to build the visual part of websites. Like JS, these are on almost every website.

Python

Python is a general-purpose programming language used in many things, like AI, robotics, and data visualization.

Git

Git is a tool for keeping track of changes in a project or a file. It helps us, developers, to not mess up our code.
